- Configuration du réseau (1)

(1) : Réseau .lan. VM connectée au réseau local. Reçoit sa propre adresse IP du routeur. VM peut communiquer avec d’autres appareils sur le réseau local. Utile pour héberger des serveurs(web). Simule mieux un environnement de réseau réel

- Création d’un mot de passe pour root

- Création de l’utilisateur et son mdp

- Partitionnement du disque(2)

- Sélection du disque SCSI

- Choix du partitionnement(3)

- Affichages des partitions et validations

(2) (3) : Permet de faire une configuration assez simple. Installation plus simple et plus rapide.
Les partitions sont déjà créer. ( /racine, swap, /home). Idéal pour les débutants.

- Configuration popularity-contest(4)

(4) : Cette configuration sert à collecter des données sur les paquets installés pour améliorer Debian.
Pas interessée par cette collecte.

- Installation programme démarrage GRUB(6)

(5) : 1 7 12-> ceux proposés pour l’installation
11-> serveur SSH, permet d’accéder à un serveur distant
10-> serveur web, utile pour hébergement web
(6) : essentiel pour démarrer correctement un ordinateur, lorsqu’ils a plusieurs OS installés.

- Choix périphériques(7)

- Fin de configurations

(7) : Installation sur disque /dev/sda (disque principal du système). Disque principal de démarrage. Permet s’assurer que le chargeur de démarrage soit bien positionné sur le disque principal, permet ainsi au système de démarrer correctement et de prendre en charge des configurations multi-boot ou autres besoins spécifiques.
